 ## Venoma Sports - Modern sport és fitnesz weboldal

Venoma Sports egy modern, reszponzív sport és fitnesz weboldal, amely inspiráló tartalmakkal, edzésprogramokkal és sportközösséggel szolgál. A weboldal célja, hogy motiválja és támogassa a felhasználókat sportos életmódjukban, legyen szó kezdőkről vagy profi sportolókról.

# Telepítés
Előfeltételek
- Node.js 18.0 vagy újabb verzió
- npm vagy yarn csomagkezelő

# Repository klónozása
```
git clone https://github.com/dajkagabi/run-venoma.git
cd run-venoma
```

# Függőségek telepítése
```
npm install
vagy
yarn install
```

#Fejlesztői szerver indítása
```
npm run dev
# vagy
yarn dev
```

# Főbb technológiák
- Next.js 16
- React 
- Tailwind CSS
- Framer Motion
- shadcn/ui
